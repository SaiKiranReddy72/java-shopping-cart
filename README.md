# Java Shopping Cart Web Application

A full-stack Java-based e-commerce web application built using
Java, JSP, Servlets, JDBC, MySQL, and Apache Tomcat.

![img alt](https://github.com/SaiKiranReddy72/java-shopping-cart/blob/2fd9d656ed075fd841fd83103778b430d57ce119/Sai-Kiran%20Electronics.jpeg)




## Features
- User Registration & Login
- Product Listing
- Add to Cart & Checkout
- Order Management
- MySQL Database Integration
- MVC Architecture

## Tech Stack
- Java (Core + Advanced)
- JSP & Servlets
- JDBC
- MySQL
- Apache Tomcat
- Maven
- HTML, CSS, Bootstrap

## Architecture
- Controller: Java Servlets
- Business Logic: Java Service Layer
- Persistence: JDBC + MySQL
- View: JSP

## How to Run
1. Configure MySQL database
2. Update DB credentials in `application.properties`
3. Build project using Maven
4. Deploy WAR on Tomcat
5. Access at `http://localhost:8080/shopping-cart-0.0.1-SNAPSHOT/`
